Bash Script

Created a npm pack with my group that allows you to get a basic MEAN stack CRUD app up and running directly off the command line. Simply npm install -g mean-scaffold and call mean-scaffold in your directory. Answer the prompts to create your models and it'll do the rest. Generated 250 downloads in its first week.
